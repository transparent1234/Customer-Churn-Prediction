# Customer-Churn-Prediction
Data Science project using Machine Learning to predict customer churn
# Customer Churn Prediction Project

## Project Overview

This is a Machine Learning project used to predict whether a customer will leave a company (churn) or stay.

The main goal of this project is to understand customer behavior using data analysis and machine learning models.

---

## Tools and Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook
* GitHub

---

## Project Steps

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Handle Missing Values
5. Exploratory Data Analysis
6. Feature Engineering
7. Train-Test Split
8. Model Training
9. Prediction
10. Model Evaluation
11. Save Model

---

## Files Included

* Customer_Churn_Prediction.ipynb
* customer_churn.csv
* churn_model.pkl
* README.md
* images/

---

## Output

The model predicts whether a customer is likely to churn or not and helps businesses improve customer retention.
